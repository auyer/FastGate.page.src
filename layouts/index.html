FastGate is a lightweight and low overhead APIGateway built with Golang using the Badger Key-Value Database and Echo Framework.

Blazing Fast

FastGate does not rely on DNS services. and redirects recieved requests to the destination adresses.

Simple Data Storage

It does not use any external database to work. It stores simple key-value data only, making it a lot faster.

Customizable

There are many customizations available to FastGate. Making it a good fit for simple projects and complicated infrastructures.

Easy to Use

You can startEvery element in kube is mobile-first and fully embraces latest and greatest tech.

Easy to Deploy

You can instantly run the pre-compiled birany, run in Docker, or compile it yourself with GO.
